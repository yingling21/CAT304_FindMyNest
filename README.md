## Features

### User Registration AND Verification

> IC-based identity verification

> Additional verification for landlords and agents

> House Listing & Search

> Create, edit, and manage rental listings

> Smart filters by location, budget, and preferences

> Map view with nearby facilities

> Rental affordability calculator

### Communication AND Review

> Real-time in-app messaging (WhatsApp-style)

> Chat list with unread message indicator

> Tenant rating and review system

### Administration Management

> Admin approval for users and listings

> System usage analytics

## Installation
Prerequisites

Node.js (v18 or above)

npm or yarn

Expo CLI

Supabase account

### Steps
#### Clone the repository
git clone https://github.com/yingling21/CAT304_FindMyNest.git

#### Navigate to project directory
cd findmynest

#### Install dependencies
npm install

#### Start Expo development server
npx expo start

Then:

1. Scan QR code using Expo Go (mobile), or

2. Run on Android Emulator / iOS Simulator
